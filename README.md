# Seahorse Multi-Omics Analysis

This project focuses on computational analysis of seahorse (Hippocampus spp.) focusing on brood pouch development and host immunity.

## Objectives
- Study gene expression in brood pouch
- Identify immune-related genes
- Perform homology modeling
- Molecular docking analysis
- Lipid-protein interaction using Lipobind

## Tools Used
- SWISS-MODEL
- AutoDock
- PyMOL
- BLAST
- Python

## Key Features
- Multi-omics integration
- Drug interaction analysis
- Structural bioinformatics
